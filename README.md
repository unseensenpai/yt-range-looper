# 🔄 YouTube Two Point Looper

[TR] YouTube videolarında belirlediğiniz iki nokta (başlangıç ve bitiş) arasında salise hassasiyetinde döngü (loop) yapmanızı sağlayan, tarayıcı kısayolları ve fare tekerleği destekli hafif bir tarayıcı eklentisi.

[EN] A lightweight browser extension that allows you to loop between two specific points (start and end) in YouTube videos with millisecond precision, featuring browser shortcuts and mouse wheel controls.

---

## 🚀 Özellikler / Features

### 🇹🇷 Türkçe
* **Salise Hassasiyeti:** `0.1` saniyelik adımlarla kusursuz döngü aralığı belirleme.
* **Akıllı Önbellek (Cache):** Girdiğiniz süreler video ID'sine göre tarayıcı hafızasında saklanır. Sayfayı yenileseniz (F5) veya videoya günler sonra dönseniz bile süreleriniz kaybolmaz.
* **Gelişmiş Kısayollar:** Video oynatılırken tek tuşla veya tıklamayla anlık süreyi hafızaya alma.
* **Fare Tekerleği (Scroll) Desteği:** Giriş kutularının üzerindeyken farenin tekerleğini kaydırarak süreyi `0.1` hassasiyetle artırıp azaltabilme.
* **Canlı Güncelleme:** Döngü açıkken süreleri değiştirdiğinizde kapat-aç yapmanıza gerek kalmadan yeni süreler anında uygulanır.

### 🇬🇧 English
* **Millisecond Precision:** Define flawless loop ranges with `0.1` second steps.
* **Smart Cache:** Saved timestamps are stored via video ID. Your settings persist even after a page refresh (F5) or days later.
* **Advanced Shortcuts:** Capture the current playback time instantly with a single keystroke or mouse click.
* **Mouse Wheel (Scroll) Support:** Hover over input fields and scroll to change values by exactly `0.1` seconds.
* **Live Update:** Modifying timestamps while looping applies changes instantly without needing to toggle the loop button.

## ⌨️ Kısayollar / Shortcuts

| Aksiyon / Action | Klavye / Keyboard | Fare / Mouse |
| :--- | :--- | :--- |
| **Başlangıç Noktası Seç** / Set Start | `[` | `Ctrl + Left Click` (On Video) |
| **Bitiş Noktası Seç** / Set End | `]` | `Shift + Left Click` (On Video) |
| **Hassas Ayar (0.1sn)** / Fine Tune | `Arrow Up / Down` | `Mouse Wheel Up / Down` (On Inputs) |

## 🛠️ Kurulum / Installation

### 🇹🇷 Türkçe
1. Bu projeyi bilgisayarınıza indirin (ZIP olarak indirebilir veya `git clone` yapabilirsiniz).
2. Tarayıcınızdan (Chrome, Edge, Opera, Brave vb.) **Eklentiler** (`chrome://extensions/`) sayfasını açın.
3. Sağ üst köşedeki **"Geliştirici Modu" (Developer Mode)** seçeneğini aktif edin.
4. Sol üstte çıkan **"Paketlenmemiş öğe yükle" (Load unpacked)** butonuna tıklayın.
5. Bu proje klasörünü seçin.
6. YouTube'a girip bir video açın ve döngünün tadını çıkarın!

### 🇬🇧 English
1. Download this project to your computer (as a ZIP or via `git clone`).
2. Open the **Extensions** page (`chrome://extensions/`) in your browser (Chrome, Edge, Opera, Brave, etc.).
3. Enable **"Developer Mode"** in the top-right corner.
4. Click the **"Load unpacked"** button in the top-left corner.
5. Select this project folder.
6. Open any YouTube video and enjoy your custom loops!
